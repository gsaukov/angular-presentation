Unit & Component Testing.

Angular's quality components!
  1. @angular/core/testing

    2. TestBed: utility that provides a testing module environment for configuring and creating instances of components, services, directives, and pipes for testing purposes. It allows you to set up the necessary dependencies and configurations required for testing components in isolation or within a test suite.

    Fixture: provides component object with methods and properties for interacting with the component instance and its associated DOM element during testing. This allows you to inspect the component's properties, trigger change detection, and interact with its DOM elements for testing and validation purposes.

  2. Jasmine.

    3. Testing framework that provides test execution, reporting, coverage test setup and orchestration.

  3. Karma.

    4. Is an utility tool that enables integration between Angular app, tests and browsers. Karma starts a development server and runs browser(s) that will perform testing activities.
