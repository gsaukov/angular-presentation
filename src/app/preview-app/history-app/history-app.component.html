Angular in a nutshell

Angular is Fun!
  1. When?

    2. Angular was first introduced by Google in 2010 as AngularJS, a JavaScript-based open-source front-end web framework. It underwent significant changes over the years and evolved into Angular, with AngularJS being the first version, Angular 2+ following suit, and so on.

  2. What it does?

    3. Angular is a platform and framework for building client-side web applications using HTML, CSS, and TypeScript/JavaScript. It provides developers with tools and libraries for building robust, scalable, and maintainable applications, including features like data binding, dependency injection, component-based architecture, routing, and more.

  3. Why it was introduced?

    4. Angular was introduced to address the challenges faced by developers in building single-page applications (SPAs) and to provide a structured framework for creating dynamic, interactive web applications. It aims to simplify the development process by promoting modularization, reusability, and maintainability of code, enabling developers to build complex applications more efficiently.

  4. What are other options?

    5. It is not a single player and there are alternatives. React(Similar), Vue(Light weight and easy but less capable), Aurelia(Light and JS based).
