Welcome to Angular Knowledge sharing! Web page technologies:
  1. DOM

    2. Static page representation or Document that includs scripts and sources. Can become very large.

  2. CSS

    3. Styles and decorations rules for the document. Can become very controversial and cause maintainability issues, performance bottlenecks too.

  3. Java Script

    4. DOM Modification monopolist. "Breathes life into HTML". It adds dynamic behavior, interactivity, and functionality to static HTML documents. It has problems too:

    • Maintenance Complexity.
    • Not optimal usage and overuse.
    • Performance.
    • Security.
    • Events pressure.
