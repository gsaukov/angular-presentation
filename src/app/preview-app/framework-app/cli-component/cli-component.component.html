Angular CLI.

CLI is supper usefull but sometimes not seen behind IDE.
  1. New! - This will generate a new project with boilerplate code and settings.

    2. Example: ng new my-project

  2. Generate! - Generate new Angular module, component, or any other Angular-related object.

    3. Example: ng generate component my-component

  3. Serve! - Start a development server with live reload capabilities using the ng serve command.

    4. Example: ng serve

  4. Build! - This compiles your application, bundles the files, and optimizes them for deployment.

    5. Example: ng build --prod

  5. Test! - Built-in support for running unit tests and end-to-end tests using tools like Karma and Protractor.

    6. Commands: ng test for running unit tests and ng e2e for running end-to-end tests.

  6. Doc! - Angular CLI can generate documentation for your Angular application using the ng doc command.

    7. Example: ng doc

  7. Add! - You can add new dependencies to your Angular project using the ng add command. This command can also automatically configure the project for the added dependency.

    8. Example: ng add @angular/material

  8. Lint! - Angular CLI integrates with TSLint to help you maintain code quality by identifying and fixing linting errors.

    9. Command: ng lint
