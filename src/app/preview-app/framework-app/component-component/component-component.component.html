Component.

Fundamental element of Angular!
  1. Template:

    2. Represents the isolated view of the component, defining its DOM structure and bindings.

  2. Class:

    3. Represents the TypeScript class associated with the component. It facilitates the manipulation of the template, performs calculations, and reacts to lifecycle events through methods known as lifecycle hooks.

  3. Styles:

    4. Consist of CSS styles that are applied to the component's template, providing visual decoration and layout.

  4. Test:

    5. Refers to the tools and methodologies used to test the behavior and functionality of the component, ensuring its correctness and reliability.
